Klenzo Apps Documentation

Welcome to the official developer and merchant documentation for Klenzo Shopify apps. Our apps are built natively for Shopify using theme-extension architecture to guarantee peak performance, zero speed loss, and 100% theme compatibility.

💡 Tip: You don't need any coding skills to install and run these apps. We automatically handle all theme integration via Shopify App Embeds.

AI Section Hub

Introduction

AI Section Hub allows you to install beautiful, pre-designed sections and templates directly into your Shopify Online Store 2.0. With over 700 sections (FAQ pages, Reels, Testimonial sliders, Volume bundles), you can customize your store without switching themes or hiring coding agencies.

1-Click Installation

To integrate AI Section Hub with your active Shopify theme:

  1. Navigate to the Shopify App Store and click Install app.
  2. Once logged in, click Activate App Embed in your admin panel.
  3. Our helper script will automatically initialize custom Liquid configurations inside your theme architecture.

Theme Editor Customization

To add sections to your homepage or product templates:

  1. Go to Shopify Admin > Online Store > Themes > Customize.
  2. In the left sidebar, click Add Section.
  3. Scroll down to the Apps tab and select the desired AI Section Hub widget.
  4. Configure styles, sizes, image assets, and titles dynamically.

Troubleshooting & Support

If you encounter layout rendering issues or CSS overlapping:

  • Ensure App Embeds is enabled in your Theme Editor settings.
  • Clear your local browser cache to reload the latest layout assets.
  • If sections appear distorted, email our support team directly at mitulzalavadiya11@gmail.com for instant assistance.

Klenzo: AI Variants

Introduction

Klenzo: AI Variants replaces standard dropdown selectors with vibrant color swatches, dynamic image buttons, and responsive size blocks. Powered by AI, it reads your variant names (e.g., "Olive Green", "Charcoal Gray") and automatically fetches matching styles.

Setup Swatches

To set up swatches on your product pages:

  1. Open the Klenzo Dashboard from your Shopify apps dashboard.
  2. Choose your swatch styles (Color bubble, image preview, custom text button).
  3. Go to the Shopify Theme Customizer, add the Klenzo Swatch block inside your Product Information section, and drag it next to the variant selector.

Customizing Color Codes

By default, Klenzo detects variant names automatically. To specify exact HEX color codes:

Open the app settings dashboard, click Color Mapping, and map custom variant names to custom colors:

{
  "Olive Green": "#6b7c3f",
  "Charcoal Gray": "#2b2b2b",
  "Ocean Blue": "#0077b6"
}
