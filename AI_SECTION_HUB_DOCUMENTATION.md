## Instagram Reels Feed Manager

The Instagram Reels Feed Manager lets you connect your public Instagram profile and display your video reels beautifully on your Shopify store — without Instagram API approval.

### How It Works
1. Enter your **public Instagram username** in the app
2. AI Section Hub scrapes your reels using a multi-method fallback strategy
3. Up to **400 reels** are fetched and stored securely as Shopify metafields
4. Select which reels to display on your store
5. The Instagram Reels section in the Section Library reads these metafields and displays them live

### Scraping Strategy (Multi-Layer Fallback)
1. Instagram Clips API
2. GraphQL API (3 document ID fallbacks)
3. Timeline feed pagination
4. HTML script tag parsing

### Storage
- Up to 400 reels stored (80 reels x 5 Shopify Shop metafield chunks)
- Slim object storage (thumbnail URL + reel ID only)
- Auto-synced to theme via Shopify metafields

### Features
- Select/deselect individual reels for display
- Add reels by direct Instagram URL
- Auto-refresh feed from Instagram
- Works with the Instagram Reels and Fully Customizable Reels Section liquid templates
- Username saved to AppInstallation metafield

**Plan Required:** Growth+

## Getting Started Guide

### Step 1: Install the App
Install **AI Section Hub** from the Shopify App Store.

### Step 2: Enable the App Embed
After installation, you'll be prompted to enable the app embed:
1. Click "Enable App Embed in Theme Editor"
2. In your Theme Editor, go to App Embeds (layer icon in left sidebar)
3. Toggle "AI Section Hub" to ON
4. Click Save
5. Return to the app and click "I have enabled App Embed, Open Dashboard"

### Step 3: Explore Sections
1. Go to **Explore Sections** in the navigation
2. Browse the catalog — use filters by category or plan
3. Click the eye icon to preview any section
4. Click **"Add to Theme"** to install it

### Step 4: Add Section to a Page
1. Open Shopify Admin → Online Store → Themes → Customize
2. Navigate to the page where you want the section
3. Click "Add Section" in the left sidebar
4. Find your section under "Apps"
5. Drag it to your desired position
6. Customize settings (colors, images, text) in the right panel
7. Click Save

### Step 5: Explore AI Studio
1. Go to **AI Studio** in the navigation
2. Try the **Brand Scanner** first — it analyzes your store and gives you brand insights
3. Use **Copy AI** to generate marketing content
4. Use **SEO Meta Tags** to optimize your pages
5. Try **Section Builder** (Growth plan) to create custom Liquid sections

## Frequently Asked Questions

### Getting Started

**What is AI Section Hub?**
AI Section Hub is a Shopify app that gives you access to a library of professionally designed theme sections. You can browse, install, and customize sections directly in your Shopify Theme Editor — no coding required.

**Which Shopify themes does it work with?**
AI Section Hub works with all Shopify Online Store 2.0 themes including Dawn, Sense, Craft, Refresh, Horizon, Debut, and most third-party themes that support app blocks.

**Do I need to know how to code?**
No. All sections are fully customizable from Shopify's native Theme Editor. Just drag, drop, and customize.

**How many sections can I install?**
There is no limit. Free plan users can install all free sections. Paid plan users unlock additional premium sections.

---

### Sections and Installation

**How do I add a section to my store?**
Go to Explore Sections → find a section → click "Add to Theme" → open the provided Theme Editor link → click "Add Section" → find your section under "Apps" → customize and save.

**Why is my installed section not showing up?**
The most common reason is that the App Embed is turned off. Go to Shopify Admin → Online Store → Themes → Customize → App Embeds (layer icon) → toggle "AI Section Hub" to ON → Save.

**Will sections slow down my store?**
No. Our sections are native Liquid code served through Shopify's CDN. They don't load external scripts and have no impact on your Lighthouse or GTmetrix scores.

**Can I use the same section on multiple pages?**
Yes. You can add any section to as many pages as you want. Each instance has its own settings.

**How do I remove a section?**
Either delete it from the Shopify Theme Editor (select → trash icon), or use the "Remove" button in AI Section Hub's section manager.

**Can I edit the section code?**
Yes. Section files are in your theme's code: Online Store → Themes → Edit Code → Sections folder.

### Technical

**Does it work with headless or custom themes?**
AI Section Hub requires Shopify Online Store 2.0 themes with native Theme Editor support. Headless themes are not supported.

**Can I use sections with multiple languages or Shopify Markets?**
Yes. Section content can be translated using Shopify's native translation tools or third-party apps like Langify or Translate & Adapt.

**What if a section shows on all pages instead of one?**
Sections added to a global template appear on all pages of that type. To restrict to one page, add it to that page's unique template in the Theme Editor.
